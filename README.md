# Spotify Clone Web Player

A Spotify-inspired web music player built with HTML, CSS, and JavaScript.

## Features
- Browse playlists and albums by mood or artist
- Play, pause, and skip songs with interactive controls
- Responsive design for mobile, tablet, and desktop
- Dynamic UI updates and smooth user experience

## Screenshots
![Screenshot](img/logo.svg)

## Getting Started
1. Clone this repository
2. Open `index.html` in your browser

## Folder Structure
- `index.html` — Main HTML file
- `style.css` — Main stylesheet
- `uttility.css` — Utility classes for styling
- `script.js` — Main JavaScript logic
- `songs/` — Music folders with info and tracks
- `img/` — Icons and images

## Responsive Design
This project is fully responsive and adapts to all screen sizes, providing a seamless experience on mobile, tablet, and desktop devices.
